📅 2026-03 👤 Junzhe Lin

问题描述

在 cmd 里面使用 pip 给 python 环境安装库的时候出现：

Failed to establish a new connection: [WinError 10061] 由于目标计算机积极拒绝，无法连接。'))'

pip 安装时出现 WinError 10061 无法连接的报错信息

解决方案

网络问题，我是因为之前使用软件默认打开了电脑的代理设置。

在任务栏搜索"代理"，打开设置，关闭代理即可。

在系统设置中关闭代理选项

然后就可以下载了：

关闭代理后 pip 成功下载安装包
